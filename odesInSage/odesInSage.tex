\begin{abstract}
This activity shows how to use Sage to solve differential equations.

\end{abstract}

\maketitle

Let's first consider the problem of solving an ODE symbolically. The following Sage cell shows how to do that:

\begin{sageCell}
t = var('t')
x = function('x')(t)
de = diff(x, t) + x - 1
gen_sol = desolve(de, [x, t])
gen_sol
\end{sageCell}
